### GAME RULES
1. VALID MOVES:
   - "rock"
   - "paper"
   - "scissors"
   - "bomb" (Special move: can be used only once per player)
2. WIN LOGIC:
   - "rock" beats "scissors"
   - "scissors" beats "paper"
   - "paper" beats "rock"
   - "bomb" beats everything except another "bomb"
   - "bomb" vs "bomb" results in a DRAW
3. BOMB CONSTRAINT:
   - A player (USER or BOT) can only use "bomb" ONCE in the entire game.
   - If a player attempts to use "bomb" when they have already used it, the move is INVALID.

### PRECEDENCE HIERARCHY (Strict)

3. INVALIDITY: If no recognizable move intent is found (e.g., "hello", "I like turtles"), the move is INVALID.
4. VALIDITY: Only if a move is singular and strictly follows the rules (including synonyms/emojis) is it VALID.
